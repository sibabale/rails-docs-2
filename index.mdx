## Key Features

- **Multi-Currency Support** - Handle transactions across multiple currencies with intelligent conversion
- **Transaction Routing** - Automatic intelligent routing between participating banks
- **Settlement Engine** - Efficient batch settlement with configurable schedules
- **Webhook Integration** - Real-time transaction status updates
- **Type-Safe SDK** - Auto-generated TypeScript client with full type definitions
- **Dashboard Analytics** - Real-time metrics and transaction insights
