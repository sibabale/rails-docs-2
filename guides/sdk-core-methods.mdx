The Rails SDK provides type-safe methods for managing users and accounts in your banking application.

## User Management

### Create User

Create a new user account with proper validation:

```typescript
import { Rails } from "@rails/sdk";

const rails = new Rails();

async function createUser() {
  const result = await rails.createUser({
    firstName: "John",
    lastName: "Doe",
    email: "john.doe@example.com",
    phoneNumber: "+1-555-0123",
    dateOfBirth: "1990-01-15",
    address: {
      street: "123 Main St",
      city: "Anytown",
      state: "CA",
      zipCode: "90210"
    }
  });

  console.log("User created:", result.userId);
  console.log("Status:", result.status);
}
```

### Get User

Retrieve user information by ID:

```typescript
const user = await rails.getUser({
  userId: "user_abc123"
});

console.log("User:", user.firstName, user.lastName);
console.log("Email:", user.email);
console.log("Status:", user.status);
```

### Update User

Update user profile information:

```typescript
const updatedUser = await rails.updateUser({
  userId: "user_abc123",
  phoneNumber: "+1-555-9999",
  address: {
    street: "456 Oak Ave",
    city: "New City",
    state: "NY",
    zipCode: "10001"
  }
});

console.log("User updated:", updatedUser.updatedAt);
```

### List Users

Get a paginated list of users:

```typescript
const users = await rails.getUsers({
  page: 1,
  pageSize: 20,
  status: "active"
});

console.log("Total users:", users.total);
console.log("Users:", users.data);
```

## Account Management

### Create Account

Create a new bank account for a user:

```typescript
const account = await rails.createAccount({
  userId: "user_abc123",
  accountType: "checking",
  currency: "USD",
  initialDeposit: 10000 // Amount in cents ($100.00)
});

console.log("Account created:", account.accountId);
console.log("Account number:", account.accountNumber);
console.log("Balance:", account.balance);
```

### Get Account Balance

Check the current balance of an account:

```typescript
const balance = await rails.getAccountBalance({
  accountId: "acc_123456"
});

console.log("Current balance:", balance.balance);
console.log("Available balance:", balance.availableBalance);
console.log("Currency:", balance.currency);
```

### Get Account Details

Retrieve complete account information:

```typescript
const account = await rails.getAccount({
  accountId: "acc_123456"
});

console.log("Account type:", account.accountType);
console.log("Status:", account.status);
console.log("Created:", account.createdAt);
```

### List User Accounts

Get all accounts for a specific user:

```typescript
const accounts = await rails.getUserAccounts({
  userId: "user_abc123"
});

console.log("User accounts:", accounts.length);
accounts.forEach(account => {
  console.log(`${account.accountType}: $${account.balance / 100}`);
});
```

### Transfer Funds

Transfer money between accounts:

```typescript
const transfer = await rails.transferFunds({
  fromAccountId: "acc_sender_123",
  toAccountId: "acc_receiver_456",
  amount: 5000, // Amount in cents ($50.00)
  currency: "USD",
  description: "Payment for services",
  idempotencyKey: "transfer_unique_123"
});

console.log("Transfer ID:", transfer.transferId);
console.log("Status:", transfer.status);
console.log("Completed at:", transfer.completedAt);
```

### Get Transaction History

Retrieve account transaction history:

```typescript
const transactions = await rails.getAccountTransactions({
  accountId: "acc_123456",
  page: 1,
  pageSize: 50,
  startDate: "2024-01-01",
  endDate: "2024-01-31"
});

console.log("Total transactions:", transactions.total);
transactions.data.forEach(txn => {
  console.log(`${txn.type}: $${txn.amount / 100} - ${txn.description}`);
});
```

## Error Handling

Handle API errors appropriately in your application:

```typescript
import * as errors from "@rails/sdk/models/errors";

try {
  const user = await rails.createUser({
    firstName: "John",
    lastName: "Doe",
    email: "existing@example.com"
  });
} catch (error) {
  if (error instanceof errors.RailsError) {
    console.log("HTTP Error:", error.statusCode, error.message);
    
    if (error.statusCode === 409) {
      console.log("User already exists with this email");
    }
  }
}
```

## Standalone Functions

Import individual functions for optimized bundle sizes:

```typescript
import { 
  createUser, 
  getUser, 
  createAccount, 
  getAccountBalance,
  transferFunds 
} from "@rails/sdk";

// Create user
const user = await createUser({
  firstName: "Alice",
  lastName: "Johnson",
  email: "alice@example.com"
});

// Create account for user
const account = await createAccount({
  userId: user.userId,
  accountType: "savings",
  currency: "USD"
});

// Check balance
const balance = await getAccountBalance({
  accountId: account.accountId
});

console.log("Account balance:", balance.balance);
```

This approach is recommended for serverless environments or applications where bundle size optimization is important.
