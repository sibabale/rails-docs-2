---
title: "Get User"
api: "GET https://api.rails.com/v1/users/{user_id}"
description: "Retrieve user details by ID"
---

Retrieves detailed information for a specific user by their unique identifier.

## Path Parameters

<ParamField path="user_id" type="string" required>
  Unique identifier for the user (UUID format)
</ParamField>

## Response

<ResponseField name="user_id" type="string">
  Unique identifier for the user
</ResponseField>

<ResponseField name="first_name" type="string">
  User's first name
</ResponseField>

<ResponseField name="last_name" type="string">
  User's last name
</ResponseField>

<ResponseField name="email" type="string">
  User's email address
</ResponseField>

<ResponseField name="phone" type="string">
  User's phone number
</ResponseField>

<ResponseField name="status" type="string">
  User status: `active`, `pending`, or `suspended`
</ResponseField>

<ResponseField name="kyc_status" type="string">
  KYC verification status: `pending`, `verified`, or `rejected`
</ResponseField>

<ResponseField name="date_of_birth" type="string">
  User's date of birth (YYYY-MM-DD format)
</ResponseField>

<ResponseField name="address" type="object">
  User's address information
  <Expandable title="Address properties">
    <ResponseField name="street" type="string">
      Street address
    </ResponseField>
    <ResponseField name="city" type="string">
      City name
    </ResponseField>
    <ResponseField name="state" type="string">
      State or province code
    </ResponseField>
    <ResponseField name="zip_code" type="string">
      Postal/ZIP code
    </ResponseField>
    <ResponseField name="country" type="string">
      Country code
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 timestamp of user creation
</ResponseField>

<ResponseField name="updated_at" type="string">
  ISO 8601 timestamp of last update
</ResponseField>

## Example

<RequestExample>

```typescript TypeScript SDK
import { Rails } from "@rails/sdk";

const rails = new Rails({
  apiKey: process.env.RAILS_API_KEY
});

const user = await rails.users.get("user_abc123def456");

console.log("User:", user.firstName, user.lastName);
console.log("Email:", user.email);
console.log("Status:", user.status);
console.log("KYC Status:", user.kycStatus);
```

```typescript Fetch API
const response = await fetch('https://api.rails.com/v1/users/user_abc123def456', {
  headers: {
    'Authorization': 'Bearer your_api_key_here',
    'Content-Type': 'application/json'
  }
});

const user = await response.json();
```

</RequestExample>

# Update User

<api>PUT https://api.rails.com/v1/users/{user_id}</api>

Updates user profile information.

## Path Parameters

<ParamField path="user_id" type="string" required>
  Unique identifier for the user (UUID format)
</ParamField>

## Request

<ParamField body="first_name" type="string">
  User's first name (2-50 characters)
</ParamField>

<ParamField body="last_name" type="string">
  User's last name (2-50 characters)
</ParamField>

<ParamField body="phone" type="string">
  Phone number in international format
</ParamField>

<ParamField body="address" type="object">
  User's address information
  <Expandable title="Address properties">
    <ParamField body="street" type="string">
      Street address
    </ParamField>
    <ParamField body="city" type="string">
      City name
    </ParamField>
    <ParamField body="state" type="string">
      State or province code
    </ParamField>
    <ParamField body="zip_code" type="string">
      Postal/ZIP code
    </ParamField>
    <ParamField body="country" type="string">
      Country code
    </ParamField>
  </Expandable>
</ParamField>

## Example

<RequestExample>

```typescript TypeScript SDK
const updatedUser = await rails.users.update("user_abc123def456", {
  phone: "+1-555-9999",
  address: {
    street: "456 Oak Avenue",
    city: "New York",
    state: "NY",
    zipCode: "10001"
  }
});

console.log("User updated:", updatedUser.updatedAt);
```

```typescript Fetch API
const response = await fetch('https://api.rails.com/v1/users/user_abc123def456', {
  method: 'PUT',
  headers: {
    'Authorization': 'Bearer your_api_key_here',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    phone: "+1-555-9999",
    address: {
      street: "456 Oak Avenue",
      city: "New York",
      state: "NY", 
      zip_code: "10001"
    }
  })
});

const updatedUser = await response.json();
```

</RequestExample>

# List Users

<api>GET https://api.rails.com/v1/users</api>

Retrieves a paginated list of users with optional filtering.

## Query Parameters

<ParamField query="page" type="number">
  Page number (default: 1)
</ParamField>

<ParamField query="page_size" type="number">
  Number of users per page (default: 20, max: 100)
</ParamField>

<ParamField query="status" type="string">
  Filter by user status: `active`, `pending`, or `suspended`
</ParamField>

<ParamField query="kyc_status" type="string">
  Filter by KYC status: `pending`, `verified`, or `rejected`
</ParamField>

<ParamField query="email" type="string">
  Search by email address (partial match)
</ParamField>

## Response

<ResponseField name="data" type="array">
  Array of user objects
</ResponseField>

<ResponseField name="pagination" type="object">
  Pagination information
  <Expandable title="Pagination properties">
    <ResponseField name="page" type="number">
      Current page number
    </ResponseField>
    <ResponseField name="page_size" type="number">
      Number of items per page
    </ResponseField>
    <ResponseField name="total" type="number">
      Total number of users
    </ResponseField>
    <ResponseField name="has_more" type="boolean">
      Whether there are more pages
    </ResponseField>
  </Expandable>
</ResponseField>

## Example

<RequestExample>

```typescript TypeScript SDK
const users = await rails.users.list({
  page: 1,
  pageSize: 20,
  status: "active",
  kycStatus: "verified"
});

console.log("Total users:", users.pagination.total);
console.log("Users on this page:", users.data.length);

users.data.forEach(user => {
  console.log(`${user.firstName} ${user.lastName} - ${user.email}`);
});
```

```typescript Fetch API
const params = new URLSearchParams({
  page: '1',
  page_size: '20',
  status: 'active',
  kyc_status: 'verified'
});

const response = await fetch(`https://api.rails.com/v1/users?${params}`, {
  headers: {
    'Authorization': 'Bearer your_api_key_here',
    'Content-Type': 'application/json'
  }
});

const result = await response.json();
```

</RequestExample>

## Error Codes

| Code | Description |
|------|-------------|
| `user_not_found` | User ID does not exist |
| `validation_failed` | Invalid request parameters |
| `access_denied` | Insufficient permissions |
| `invalid_page` | Page number is invalid or out of range |
