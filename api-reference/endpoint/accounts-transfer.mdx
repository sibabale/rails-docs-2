---
title: "Transfer Funds"
api: "POST https://api.rails.com/api/v1/accounts/{id}/transfer"
description: "Transfer funds from one account to another"
---

Transfers funds from one account to another.

## Path Parameters

<ParamField path="id" type="string" required>
  Source account ID (UUID format)
</ParamField>

## Request

<ParamField body="to_account_id" type="string" required>
  Destination account ID (UUID format)
</ParamField>

<ParamField body="amount" type="number" required>
  Amount to transfer (must be greater than 0 and not exceed source account balance)
</ParamField>

<ParamField body="description" type="string">
  Optional description for the transaction
</ParamField>

## Response

<ResponseField name="from_account" type="object">
  Updated source account with new balance
</ResponseField>

<ResponseField name="to_account" type="object">
  Updated destination account with new balance
</ResponseField>

<ResponseField name="transaction" type="object">
  Transaction record for the transfer
</ResponseField>

## Example

<RequestExample>

```typescript TypeScript SDK
import { Rails } from "@rails/sdk";

const rails = new Rails({
  apiKey: process.env.RAILS_API_KEY
});

const result = await rails.accounts.transfer("123e4567-e89b-12d3-a456-426614174000", {
  toAccountId: "789e1234-e89b-12d3-a456-426614174000",
  amount: 250.00,
  description: "Transfer to savings"
});

console.log("Transfer completed:", result.transaction.id);
console.log("From balance:", result.fromAccount.balance);
console.log("To balance:", result.toAccount.balance);
```

```typescript Fetch API
const response = await fetch('https://api.rails.com/api/v1/accounts/123e4567-e89b-12d3-a456-426614174000/transfer', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer your_api_key_here',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    to_account_id: "789e1234-e89b-12d3-a456-426614174000",
    amount: 250.00,
    description: "Transfer to savings"
  })
});

const result = await response.json();
```

## Error Codes

| Code | Description |
|------|-------------|
| `validation_error` | Amount must be greater than zero |
| `insufficient_funds` | Source account balance is less than transfer amount |
| `account_not_found` | Source or destination account does not exist |
| `account_suspended` | Source or destination account is suspended |
| `account_closed` | Source or destination account is closed |
| `same_account_transfer` | Cannot transfer to the same account |
| `currency_mismatch` | Source and destination accounts have different currencies |
| `unauthorized` | Invalid or missing API key |
