---
title: "Get Account"
api: "GET https://api.rails.com/api/v1/accounts/{id}"
description: "Retrieve a specific account by ID"
---

Retrieves detailed information for a specific account by its unique identifier.

## Path Parameters

<ParamField path="id" type="string" required>
  Unique identifier for the account (UUID format)
</ParamField>

## Response

<ResponseField name="id" type="string">
  Unique identifier for the account (UUID)
</ResponseField>

<ResponseField name="account_number" type="string">
  Auto-generated account number
</ResponseField>

<ResponseField name="account_type" type="string">
  Type of account: `checking` or `saving`
</ResponseField>

<ResponseField name="user_id" type="string">
  ID of the account owner
</ResponseField>

<ResponseField name="balance" type="number">
  Current account balance (decimal)
</ResponseField>

<ResponseField name="currency" type="string">
  Currency code (e.g., `USD`)
</ResponseField>

<ResponseField name="status" type="string">
  Account status: `active`, `suspended`, or `closed`
</ResponseField>

<ResponseField name="organization_id" type="string">
  Organization ID (if applicable)
</ResponseField>

<ResponseField name="environment" type="string">
  Environment: `production` or `sandbox`
</ResponseField>

<ResponseField name="admin_user_id" type="string">
  Admin user ID (if applicable)
</ResponseField>

<ResponseField name="user_role" type="string">
  User role (if applicable)
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 timestamp of account creation
</ResponseField>

<ResponseField name="updated_at" type="string">
  ISO 8601 timestamp of last update
</ResponseField>

## Example

<RequestExample>

```typescript TypeScript SDK
import { Rails } from "@rails/sdk";

const rails = new Rails({
  apiKey: process.env.RAILS_API_KEY
});

const account = await rails.accounts.get("123e4567-e89b-12d3-a456-426614174000");

console.log("Account Type:", account.accountType);
console.log("Balance:", account.balance);
console.log("Status:", account.status);
```

```typescript Fetch API
const response = await fetch('https://api.rails.com/api/v1/accounts/123e4567-e89b-12d3-a456-426614174000', {
  headers: {
    'Authorization': 'Bearer your_api_key_here',
    'Content-Type': 'application/json'
  }
});

const account = await response.json();
```

</RequestExample>

## Error Codes

| Code | Description |
|------|-------------|
| `account_not_found` | Account ID does not exist |
| `unauthorized` | Invalid or missing API key |
| `invalid_uuid` | Invalid UUID format for account ID |
