---
title: "Close Account"
api: "DELETE https://api.rails.com/api/v1/accounts/{id}"
description: "Close a specific account"
---

Closes a specific account. This action sets the account status to `closed`.

## Path Parameters

<ParamField path="id" type="string" required>
  Unique identifier for the account (UUID format)
</ParamField>

## Response

<ResponseField name="id" type="string">
  Unique identifier for the account (UUID)
</ResponseField>

<ResponseField name="account_number" type="string">
  Account number
</ResponseField>

<ResponseField name="account_type" type="string">
  Type of account: `checking` or `saving`
</ResponseField>

<ResponseField name="user_id" type="string">
  ID of the account owner
</ResponseField>

<ResponseField name="balance" type="number">
  Final account balance (decimal)
</ResponseField>

<ResponseField name="currency" type="string">
  Currency code (e.g., `USD`)
</ResponseField>

<ResponseField name="status" type="string">
  Account status: `closed`
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 timestamp of account creation
</ResponseField>

<ResponseField name="updated_at" type="string">
  ISO 8601 timestamp of closure
</ResponseField>

## Example

<RequestExample>

```typescript TypeScript SDK
import { Rails } from "@rails/sdk";

const rails = new Rails({
  apiKey: process.env.RAILS_API_KEY
});

const account = await rails.accounts.close("123e4567-e89b-12d3-a456-426614174000");

console.log("Account closed:", account.id);
console.log("Status:", account.status);
```

```typescript Fetch API
const response = await fetch('https://api.rails.com/api/v1/accounts/123e4567-e89b-12d3-a456-426614174000', {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer your_api_key_here',
    'Content-Type': 'application/json'
  }
});

const account = await response.json();
```

</RequestExample>

## Error Codes

| Code | Description |
|------|-------------|
| `account_not_found` | Account ID does not exist |
| `unauthorized` | Invalid or missing API key |
| `account_has_balance` | Cannot close account with non-zero balance |
| `account_already_closed` | Account is already closed |
