---
title: "Deposit Funds"
api: "POST https://api.rails.com/api/v1/accounts/{id}/deposit"
description: "Deposit funds into an account"
---

Deposits funds into a specific account.

## Path Parameters

<ParamField path="id" type="string" required>
  Unique identifier for the account (UUID format)
</ParamField>

## Request

<ParamField body="amount" type="number" required>
  Amount to deposit (must be greater than 0)
</ParamField>

<ParamField body="description" type="string">
  Optional description for the transaction
</ParamField>

## Response

<ResponseField name="account" type="object">
  Updated account object with new balance
</ResponseField>

<ResponseField name="transaction" type="object">
  Transaction record for the deposit
</ResponseField>

## Example

<RequestExample>

```typescript TypeScript SDK
import { Rails } from "@rails/sdk";

const rails = new Rails({
  apiKey: process.env.RAILS_API_KEY
});

const result = await rails.accounts.deposit("123e4567-e89b-12d3-a456-426614174000", {
  amount: 500.00,
  description: "Salary deposit"
});

console.log("New balance:", result.account.balance);
console.log("Transaction ID:", result.transaction.id);
```

```typescript Fetch API
const response = await fetch('https://api.rails.com/api/v1/accounts/123e4567-e89b-12d3-a456-426614174000/deposit', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer your_api_key_here',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    amount: 500.00,
    description: "Salary deposit"
  })
});

const result = await response.json();
```

</RequestExample>

## Error Codes

| Code | Description |
|------|-------------|
| `validation_error` | Amount must be greater than zero |
| `account_not_found` | Account ID does not exist |
| `account_suspended` | Account is suspended and cannot receive deposits |
| `account_closed` | Account is closed and cannot receive deposits |
| `unauthorized` | Invalid or missing API key |
